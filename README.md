# 🍲 Home Recipe Website

A dynamic and user-friendly **Home Recipe Website** that allows users to explore recipes, place orders, and provide feedback.  
This project is designed to simulate a real-world food ordering platform with multiple pages and basic data handling.

---

## 📌 Features

- 🏠 Home Page – Introduction to the website and featured recipes  
- 📖 Recipes Page – Displays different recipes with details  
- 📍 Address Page – Shows business or delivery address  
- 📩 Contact Page – Users can reach out for queries  
- 💬 Feedback Page – Users can submit their feedback  
- 🛒 Order Page – Users can place food orders  
- 📊 Order Details – Retrieve and display up to 10 order records  

---

## 🚀 How It Works

1. Users visit the website and browse recipes  
2. They can place an order through the **Order Page**  
3. Order details are stored and can be retrieved (up to 10 records)  
4. Users can also submit feedback through the **Feedback Page**  

---

## ▶️ How to Run

1. Download or clone this repository  
2. Open the project folder  
3. Run `index.html` in your browser  

*(If backend is used, run using XAMPP/WAMP server)*
